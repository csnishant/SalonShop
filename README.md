

## 🚀 How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/csnishant/salonshop
```
2. Navigate to the client directory:
```bash
cd client
```
3. Install dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```
