# Nexus Clinic – Premium Medical Web Experience

Welcome to the **Nexus Clinic** project! This is a **high-end digital experience for a medical clinic** in Kuala Lumpur, Malaysia, designed with iOS-inspired aesthetics and premium medical UI.

All frontend code for this project is located in the [`client`](client) directory.

---

## 🌟 Project Overview

The goal of this project is to transform a standard clinic website into a **Stealth-Luxury digital experience** that is:

- Clean and distraction-free  
- Responsive on both mobile and desktop  
- Conversion-focused with interactive CTAs  
- Trust-building with real-time doctor availability and accreditation badges  

---

## 🧱 Key Features

### 1. Parallax Hero Section
- **Glassmorphism**: Floating badges and glass buttons for depth  
- **Scroll Animation**: Smooth animations for immersive scrolling  
- **Dynamic Gradients**: Emerald and blue glowing orbs on a dark theme for a luxurious feel  

### 2. About / Nexus Standard
- **Bento-style stats**: Minimalist cards showing patients, expertise, and success rates  
- **Animated counters**: Numbers animate from zero on scroll to build authority  
- **Feature accordion**: Interactive vertical selector for clinic protocols  

### 3. Signature Treatments
- **Hover effects**: Cards expand and reveal CTA buttons  
- **Trust badges**: FDA-approved and MOH-accredited badges for confidence  
- **Color-coded gradients**: Subtle overlays for different treatment types  

### 4. Transformation Stories / Testimonials
- **Smooth spring animations** for switching testimonials  
- **Verified patient ratings and Google reviews**  
- **Layered “ghost cards”** for modern depth effect  

### 5. Doctor Profiles
- **Circular frames** with pulsing status indicators  
- **Grayscale images by default**, color on hover for focus

## 🚀 How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/csnishant/nexus-clinic.git
```
2. Navigate to the client directory:
```bash
cd nexus-clinic/client
```
3. Install dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```
